# 🎭 Искусство жёлтых заголовков: Интрига без спойлеров

## Проблема: Заголовки-спойлеры

### ❌ ЧТО БЫЛО (плохо):
```
Студент убил старуху-процентщицу топором
Замужняя дама бросилась под поезд  
Дворянин убил лучшего друга на дуэли
```

**Проблема:** Весь сюжет раскрыт! Интерес пропал.

### ✅ ЧТО СТАЛО (хорошо):
```
Студент скрывал страшную тайну: что довело его до отчаяния?
Роковая страсть довела светскую даму до трагедии
Дворянин не простил оскорбления: дуэль закончилась трагедией
```

**Результат:** Интрига! Хочется узнать, что же произошло.

---

## 📋 Золотое правило жёлтой прессы

> **ИНТРИГУЙ, НЕ РАСКРЫВАЙ!**
> 
> Заголовок должен вызывать вопрос "Что же произошло?",  
> а не отвечать на него.

---

## 🎨 Приёмы создания интриги

### 1. Недосказанность

**Вместо прямого действия → намёк на последствие**

| ❌ Спойлер | ✅ Интрига |
|-----------|-----------|
| Студент убил старуху | Студент совершил страшное преступление |
| Женщина застрелилась | Женщина приняла роковое решение |
| Офицер проиграл в карты | Офицер потерял всё за одну ночь |

### 2. Эмоции и последствия

**Фокус на чувствах, а не на фактах**

| ❌ Спойлер | ✅ Интрига |
|-----------|-----------|
| Замужняя бросилась под поезд | Роковая страсть довела даму до отчаяния |
| Дворянин застрелил друга | Дворянин не простил оскорбления |
| Князь сошёл с ума | Трагедия разрушила князя |

### 3. Вопросы и многоточия

**Вовлечение читателя**

```
✅ Что скрывал молодой человек?
✅ Чем обернулась роковая встреча?
✅ Он думал, что тайна останется нераскрытой, но...
✅ Трагедия в высшем свете: подробности...
```

### 4. Двойной смысл

**Интрига через неоднозначность**

```
✅ Студент заплатил страшную цену за свои убеждения
   (что случилось? убили его? или он кого-то убил?)

✅ Любовный треугольник довёл купца до края
   (что он сделал? убил? сошёл с ума? разорился?)

✅ Офицер пересёк роковую черту
   (какую? дезертировал? убил? изменил?)
```

## 📐 Формулы интригующих заголовков

### Формула 1: "[Роль] + [что-то страшное] + вопрос"
```
Студент скрывал страшную тайну: что довело его до отчаяния?
Офицер совершил непростительное: чем обернулась дуэль?
Купец пошёл на отчаянный шаг: что произошло дальше?
```

### Формула 2: "[Ситуация] довела [роль] до [последствие]"
```
Роковая страсть довела светскую даму до трагедии
Любовный треугольник довёл купца до страшного поступка
Долги довели помещика до отчаянного решения
```

### Формула 3: "[Роль] + [действие] + двоеточие + недосказанность"
```
Дворянин не простил оскорбления: дуэль закончилась трагедией
Офицер похитил княжну: скандал потряс общество
Чиновник провернул аферу: как его разоблачили?
```

### Формула 4: "[Событие] изменило всё"
```
Война изменила жизнь офицера навсегда: откровения из плена
Одна ночь разрушила карьеру дворянина: что произошло?
Встреча с незнакомкой обернулась трагедией для студента
```

### Формула 5: "Он не знал, что..."
```
Он думал, что тайна останется нераскрытой, но...
Она не знала, что роковая встреча изменит всё
Офицер не ожидал, что дуэль обернётся так страшно
```

## 🎯 Ключевые слова для интриги

### Недосказанность
- довела до... (отчаяния, трагедии, края)
- привела к... (страшным последствиям, скандалу)
- обернулась... (трагедией, катастрофой)
- закончилась... (ужасно, неожиданно)
- принял... (роковое решение, страшный выбор)

### Вопросы
- Что произошло?
- Чем закончилось?
- Как это случилось?
- Что скрывал?
- Чем обернулось?

### Эмоциональные слова
- роковой, страшный, ужасный, трагический
- шокирующий, невероятный, скандальный
- тайный, скрытый, запретный
- отчаянный, безумный, дерзкий

### Последствия (вместо действий)
- совершил страшное (вместо "убил")
- принял роковое решение (вместо "застрелился")
- довёл до трагедии (вместо "убил себя")
- пересёк черту (вместо "изменил/предал")
- заплатил страшную цену (вместо "умер")

## ⚙️ Обновлённая система оценки

### Новые критерии (добавлены к существующим)

**БОНУСЫ (+):**
- Использует вопросы: +0.15
- Недосказанность ("довела до...", "обернулась..."): +0.15
- Эмоциональные слова (роковой, страшный): +0.10
- Двоеточие для интриги: +0.05

**ШТРАФЫ (-):**
- Раскрывает концовку (убил, застрелился, бросилась): -0.40
- Слишком прямолинейно (нет интриги): -0.20

## 💡 Практические советы

### 1. Два уровня информации

**Уровень 1 (даём):** Кто и что делает  
**Уровень 2 (скрываем):** Как именно и чем закончилось

```
✅ Студент совершил страшное [ЧТО?] преступление [КАКОЕ?]
✅ Дама приняла роковое [ЧТО?] решение [КАКОЕ?]
✅ Офицер пересёк роковую [КАКУЮ?] черту
```

### 2. Тест на спойлер

Задай вопрос: **"Если я прочитаю только заголовок, останется ли интерес читать книгу?"**

- Если НЕТ → спойлер, переделывай
- Если ДА → отлично, интрига работает

### 3. Замени глаголы

| ❌ Конкретный глагол | ✅ Эвфемизм |
|---------------------|-------------|
| убил | совершил страшное, пересёк черту |
| застрелился | принял роковое решение |
| бросилась под поезд | довела до трагедии |
| проиграл в карты | потерял всё за одну ночь |
| изменила мужу | предалась роковой страсти |
